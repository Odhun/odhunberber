# İptal Edilen Fikirler

## Amaç
Daha önce değerlendirilip reddedilen fikirleri kaydetmek — farklı AI araçlarının aynı fikri tekrar tekrar önermesini engellemek.

## Kapsam
Ürün, mimari veya süreç önerileri.

## Kullanım Şekli
Kullanıcı bir öneriyi "hayır, bunu yapmayalım" diye reddettiğinde buraya kaydedilir.

## Kayıtlar
Henüz kayıt yok. Bu dosya `_project-system` kurulumu sırasında (2026-07-15) oluşturuldu; geriye dönük olarak "reddedildiği doğrulanmış" bir fikir tespit edilmedi (bu tür kararlar önceden başka bir yerde belgelenmemişti).

## Kayıt Formatı
```
### YYYY-MM-DD — [Fikir Başlığı]
- Fikir:
- Neden değerlendirildi:
- Neden iptal edildi:
- Hangi şartlarda yeniden değerlendirilebilir:
```
